# Positioning

## Hero Hook

Your AI agent remembers everything. Can you prove it forgot?

## Product Answer

Orenyl is agent memory with deletion guarantees.

## Proof Claim

When data is removed, every derived insight is traced, invalidated, and recomputed so deleted information does not resurface.

## Approved Supporting Phrases

- production-minded early production / public beta
- deterministic event, fact, and lineage model
- deletion proof and verification proof
- MCP-native integration
- local-first evaluation and demos

## Phrases To Avoid In The First Screen

- production-minded governed memory MCP server
- deterministic deletion as the lead line
- auditable lineage as the opening claim
- enterprise-certified
- only solution
